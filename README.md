SATDS is a modern AI-powered security monitoring dashboard designed to simulate real-time threat detection and surveillance operations. The system integrates multiple security modules, including behavior analysis, facial recognition, perimeter monitoring, NLP-based threat detection, and alert management. Built with HTML, CSS, and JavaScript, it features a responsive dark-themed interface with live camera feeds, threat scoring, system logs, and interactive controls. SATDS demonstrates how AI technologies such as YOLO, OpenCV, BERT, PyTorch, and TensorFlow can be combined to enhance public safety, detect suspicious activities, and support intelligent security decision-making.
